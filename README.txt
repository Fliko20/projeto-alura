 Bem vindo ao decodificador e codificador de mensagem!! obrigado por ter aberto este arquivo!

 Para poder codificar uma palavra, escreva no quadrado azul e após isso clique em codificar.

 Para decodificar uma palavra, siga o mesmo processo anterior e clique em decodificar.

 OBS: 
A letra "e" é convertida para "enter"
A letra "i" é convertida para "imes"
A letra "a" é convertida para "ai"
A letra "o" é convertida para "ober"
A letra "u" é convertida para "ufat"

 E não utilize letras maiúsculas e caracteres especiais.
